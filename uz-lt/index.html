So‘rash uchun so‘rama, to‘g‘ridan-to‘g‘ri so‘ra

Vaqti-vaqti bilan, men mavjud bo'lgan onlayn chatlarda kimdir(lar), quyidagiga o‘xshash gapni aytadi:

User:

Bu yerda Java bo‘yicha ekspertlar bormi?

Bu yomon uslub, bir nechta sabablarga ko‘ra. Aslida bu yerda odam haqiqatda quyidagini so‘rayapti:

User:

Bu yerda Java bo‘yicha ekspertlar bormi, ular mening muammomni ko‘rib chiqishga rozi bo‘ladimi — u muammo aslida Java’ga aloqador bo‘lmasligi ham mumkin, yoki Java haqida hech narsa bilmaydigan odam ham javob bera oladigan savol bo‘lsa ham?

Bilimga ega bo‘lgan odamlar nima uchun buni tan olmasligiga juda ko‘p sabablar bor. Siz o‘ylaganingizdan ko‘ra ko‘proq narsani so‘rayapsiz.

Siz odamlardan mas’uliyat olishni so‘rayapsiz. Ularning o‘z qobiliyatlariga bo‘lgan ishonchini shubha ostiga qo‘yyapsiz. Shu bilan birga, boshqa odamlarni ham asossiz ravishda chetga chiqarayapsiz. Men ko‘pincha hech qachon ishlatmagan tillar yoki kutubxonalar bo‘yicha ham savollarga javob beraman, chunki javoblar (dasturcha nuqtayi nazardan) oddiy mantiqqa asoslangan bo‘ladi.

Yoki bu quyidagicha ham qabul qilinishi mumkin:

Foobar123:

Menda Java haqida savol bor, lekin uni aniq so‘zlar bilan ifodalashga erinayapman, agar kanalda javob bera oladigan kimdir bo‘lmasa

Bu esa oddiy dangasalik. Agar siz o‘z muammolaringizni hal qilish uchun mehnat qilishga tayyor bo‘lmasangiz, nega biz qilishimiz kerak?

Yechim — “so‘rash uchun so‘rash” emas, balki to‘g‘ridan-to‘g‘ri so‘rash. Kanalda bekor o‘tirib, faqat vaqti-vaqti bilan chatga ko‘z tashlab turadigan odam sizning “so‘rashga ruxsat so‘rashingiz”ga javob bermasligi mumkin, ammo muammoning aniq tavsifi uni qiziqtirib, javob berishga undashi mumkin.

Xulosa qilib aytganda, “Bu yerda Java ekspertlari bormi?” deb so‘ramang, balki shunday so‘rang: “Java yordamida [muammo]ni qanday hal qilsam bo‘ladi va [boshqa muhim ma’lumotlar]?”

Shunga o‘xshash boshqa muammolar: XY muammosi, No Hello. Qo‘shimcha o‘qish uchun: Yaxshi savolni qanday berish kerak?, yoki agar vaqtingiz ko‘proq bo‘lsa: Aqlli tarzda savol berish usuli.
