Não peça para perguntar, apenas pergunte

De vez em quando, nas salas de bate papo online em que eu participo, aparece alguém falando algo parecido com:

Foobar123:

Alguém que entende de Java por aí?

Esse é um jeito ruim, por vários motivos. O que a pessoa está realmente perguntando é:

Foobar123:

Alguém por aí que entende de Java está disposto a se comprometer a olhar o meu problema, qualquer ele que seja, mesmo se ele não for exatamente relacionado com Java ou mesmo se alguém que não sabe nada de Java puder responder a minha pergunta?

Existem várias razões pelas quais as pessoas que TÊM o conhecimento não admitiriam que tem. E ao perguntar, você está pedindo por mais do que você acha que está pedindo.

Você está pedindo para as pessoas assumirem a responsabilidade. Você está questionando a confiança das pessoas em suas próprias habilidades. Você também está trancando as outras pessoas para fora. Eu geralmente respondo perguntas relacionadas a linguagens ou bibliotecas que eu nunca usei, porque as respostas são (de uma forma geral na programação) senso comum.

Ou então, isso pode ser visto como..

Foobar123:

Eu tenho uma dúvida sobre Java, mas eu sou muito preguiçoso para formalizar ela em palavras a menos que haja alguém no canal que esteja disposto a respondê-la

..simplesmente preguiça. Se você não está disposto a trabalhar para resolver o seu problema, porque eu estaria?

A solução é não pedir para perguntar, mas simplesmente perguntar. Alguém que esteja ocioso no canal e apenas de vez em quando aparece para ver o que está acontecendo dificilmente vai responder ao seu "pedido de pergunta", mas a descrição do seu problema real talvez desperte o interesse e ele dedique algum tempo para responder.

Então, para resumir, não pergunte "Alguém que conhece de Java por aí?", em vez disso, pergunte "Como eu faço [problema] com Java e [outras informações relevantes]?"

Outros problemas similares: O problema XY, Não diga olá. Leitura aprofundada: Como eu posso fazer uma boa pergunta?, ou se você tem mais tempo: Como Perguntar de Forma Inteligente.
