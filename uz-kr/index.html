Сўраш учун сўрама, тўғридан-тўғри сўра

Вақти-вақти билан, мен иштирок этадиган онлайн чат хоналарида кимдир кириб, қуйидагига ўхшаш гапни айтади:

Foobar123:

Бу ерда Java бўйича экспертлар борми?

Бу ёмон услуб, бир неча сабабларга кўра. Аслида бу ерда инсон ҳақиқатан қуйидагини сўраяпти:

Foobar123:

Бу ерда Java бўйича экспертлар борми, улар менинг муаммомни кўриб чиқишга рози бўладими — у муаммо аслида Java’га боғлиқ бўлмаслиги ҳам мумкин, ёки Java ҳақида ҳеч нарса билмайдиган одам ҳам жавоб бера олиши мумкин бўлган савол бўлса ҳам?

Билимга эга бўлган одамлар нима учун буни тан олмаслигига жуда кўп сабаблар бор. Сиз сўраётганингиздан кўра кўпроқ нарсани талаб қилаяпсиз.

Сиз одамлардан жавобгарлик олишни сўраяпсиз. Улар ўз қобилиятларига бўлган ишончини шубҳа остига қўяяпсиз. Шу билан бирга, бошқа одамларни ҳам асоссиз равишда четга чиқаряпсиз. Мен кўпинча ҳеч қачон ишлатмаган тиллар ёки кутубхоналар бўйича ҳам саволларга жавоб бераман, чунки жавоблар (дастурчи нуқтаи назаридан) оддий мантиққа асосланган бўлади.

Ёки бу қуйидагича ҳам қабул қилиниши мумкин:

Foobar123:

Менда Java ҳақида савол бор, лекин уни аниқ сўзлар билан ифодалашга эриняпман, агар каналда жавоб бера оладиган кимдир бўлмаса

Бу эса оддий дангасалик. Агар сиз ўз муаммоларингизни ҳал қилиш учун меҳнат қилишга тайёр бўлмасангиз, нега биз қилишимиз керак?

Ечим — «сўраш учун сўраш» эмас, балки тўғридан-тўғри сўраш. Каналда бекор ўтириб, фақат вақти-вақти билан чатга кўз ташлаб турадиган одам сизнинг «сўрашга рухсат сўрашингиз»га жавоб бермаслиги мумкин, аммо муаммонинг аниқ тасвири уни қизиқтириб, жавоб беришга ундаши мумкин.

Хулоса қилиб айтганда, «Бу ерда Java экспертлари борми?» деб сўраманг, балки шундай сўранг: «Java ёрдамида [муаммо]ни қандай ҳал қилсам бўлади ва [бошқа муҳим маълумотлар]?»

Шунга ўхшаш бошқа муаммолар: XY муаммоси, No Hello. Қўшимча ўқиш учун: Яхши саволни қандай бериш керак?, ёки агар вақтингиз кўпроқ бўлса: Ақлли тарзда савол бериш усули.
